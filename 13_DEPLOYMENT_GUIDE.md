# 13_DEPLOYMENT_GUIDE.md

# 13. 導入・環境構築手順

## 13.1 目的

本書は、本アプリケーションを新規環境へ導入するための手順を示す。

------------------------------------------------------------------------

## 13.2 前提条件

-   Python 3.12 以上
-   Git
-   Gmail アカウント（IMAP有効）
-   Gmail アプリケーションパスワード
-   WordPress 管理者権限
-   WordPress Application Password

------------------------------------------------------------------------

## 13.3 プロジェクト取得

``` bash
git clone <repository>
cd project
```

------------------------------------------------------------------------

## 13.4 仮想環境作成

``` bash
python -m venv .venv
```

### 有効化

macOS / Linux

``` bash
source .venv/bin/activate
```

Windows

``` cmd
.venv\Scripts\activate
```

------------------------------------------------------------------------

## 13.5 ライブラリインストール

``` bash
pip install -r requirements.txt
```

------------------------------------------------------------------------

## 13.6 ディレクトリ作成

``` text
config/
history/
images/
logs/
temp/
```

------------------------------------------------------------------------

## 13.7 設定ファイル配置

`config/config.yaml`

-   Gmail設定
-   WordPress設定
-   ログ設定
-   投稿設定

`config/template.html`

-   投稿テンプレートを配置

------------------------------------------------------------------------

## 13.8 動作確認

``` bash
python main.py
```

確認項目

-   Gmail接続成功
-   添付画像取得
-   WordPress接続成功
-   下書き記事作成
-   ログ出力

------------------------------------------------------------------------

## 13.9 本番導入

推奨手順

1.  テスト環境で動作確認
2.  設定ファイル確認
3.  バックアップ取得
4.  本番環境へ配置
5.  下書き投稿で確認
6.  運用開始

------------------------------------------------------------------------

## 13.10 アップデート

1.  バックアップ取得
2.  最新ソース取得
3.  requirements 更新
4.  動作確認
5.  本番反映

------------------------------------------------------------------------

## 13.11 トラブルシューティング

  症状                   確認項目
  ---------------------- ----------------------
  Gmail接続不可          IMAP・認証
  WordPress接続不可      Application Password
  画像投稿失敗           REST API
  テンプレート読込失敗   配置場所
