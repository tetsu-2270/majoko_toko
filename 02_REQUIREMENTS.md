# 02_REQUIREMENTS.md

# 要件定義

## 1. 目的

Gmailへ送信されたメールを自動解析し、WordPressへ記事を自動投稿する。

------------------------------------------------------------------------

# 2. 機能要件

## 2.1 Gmail受信

-   未読メールのみ取得
-   「投稿済み」ラベルが付いていないメールのみ対象
-   Gmail APIを利用する

## 2.2 件名解析

件名形式

    Title No12

### ルール

-   Titleを記事タイトルとして利用
-   Noが存在する場合はその番号を利用
-   Noが無い場合はWordPressから同一タイトルの最新記事を検索し、最新番号+1を採番する

## 2.3 メール本文

本文形式

    カテゴリー：4コマ

    投稿日：2026/07/10 20:00

### 必須

-   カテゴリー

### 任意

-   投稿日（未指定なら即時公開）

## 2.4 添付画像

-   添付画像をimagesフォルダへ保存
-   ファイル名は変更しない
-   IMG9999→IMG0001を考慮して並び替える
-   並び替え後の順番でWordPressへアップロード

## 2.5 HTML生成

template.htmlを読み込む。

### 前回記事

    <!-- 前回の話 -->

を前回記事リンクへ置換する。

### 本文

    <!-- 以下本文 -->

を画像HTMLへ置換する。

## 2.6 WordPress

-   Application Password認証
-   REST API利用
-   カテゴリーが存在しない場合は自動作成
-   1枚目画像をアイキャッチ画像に設定
-   投稿日指定時は予約投稿

## 2.7 投稿成功後

成功時のみ

-   Gmail既読
-   投稿済みラベル付与
-   投稿履歴DB登録

失敗時は何も変更しない。

------------------------------------------------------------------------

# 3. 非機能要件

-   Python3.12以上
-   SQLite利用
-   logging利用
-   型ヒント必須
-   pytest対応
-   1モジュール1責務

------------------------------------------------------------------------

# 4. 除外事項

初期バージョンでは対象外

-   AI文章生成
-   AI画像生成
-   SNS同時投稿
-   GUI
-   Docker
