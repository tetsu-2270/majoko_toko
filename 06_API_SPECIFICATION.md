# 6. API設計

## 6.1 概要

本システムは以下の外部APIを利用する。

-   Gmail(IMAP)
-   WordPress REST API

------------------------------------------------------------------------

## 6.2 Gmail接続

### 認証方式

-   Gmail アプリケーションパスワード
-   IMAP over SSL

### 接続先

  項目   値
  ------ ----------------
  Host   imap.gmail.com
  Port   993
  SSL    有効

### 処理内容

-   未読メール取得
-   添付画像取得
-   Message-ID取得
-   処理後に既読化
-   任意ラベル付与

------------------------------------------------------------------------

## 6.3 WordPress認証

### 認証方式

Application Password

Authorization Header

Basic Authentication

------------------------------------------------------------------------

## 6.4 メディアアップロード

### エンドポイント

POST /wp-json/wp/v2/media

### 処理

-   元画像アップロード
-   Media ID取得
-   URL取得

------------------------------------------------------------------------

## 6.5 記事投稿

POST /wp-json/wp/v2/posts

送信項目

-   title
-   content
-   status
-   categories
-   featured_media

戻り値

-   Post ID
-   URL

------------------------------------------------------------------------

## 6.6 カテゴリ取得

GET /wp-json/wp/v2/categories

存在しない場合

POST /wp-json/wp/v2/categories

------------------------------------------------------------------------

## 6.7 アイキャッチ画像

featured_mediaへMedia IDを設定

------------------------------------------------------------------------

## 6.8 エラー処理

  HTTP   対応
  ------ ----------------
  200    成功
  201    作成成功
  400    入力エラー
  401    認証エラー
  403    権限不足
  404    対象なし
  429    リトライ
  500    ログ出力後終了

------------------------------------------------------------------------

## 6.9 リトライ

対象

-   通信タイムアウト
-   429
-   一時的な500

回数

3回

指数バックオフを採用する。
